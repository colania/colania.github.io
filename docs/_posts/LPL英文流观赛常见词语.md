# LPL英文流观赛常见词语

兴趣会让人进步更快，所以最近开始看起了英文流解说的LPL比赛。一开始必然会有很多专业性的词语听不懂，以及游戏解说们的口速都是飞快的。。。 所以看起来还是比较难。但是好在自身能分析游戏战况，能大致清楚当前发生了什么事情。所以了解一些常用词语，以及听懂一些关键词，对于听懂英文流解说还是有很大帮助的。`听懂关键词+ 自己对游戏的了解`  大致和 解说想表达的意思差不多多少。
